# S3: TPH Structural-Gradient Thrust

From "Aetherion, The Jumper" - Chapter II, Section 2.3

## Overview
Models thrust from dynamically modulating internal hierarchy L(x).

## Key Equations
```
f_α = κ_eff × ε_vac × ∂α/∂x × ln(L/L₀)  (temporal)
f_L = κ_eff × ε_vac × α × (1/L) × ∂L/∂x  (geometric)
```

## Hybrid Strategy
- Slow α-shaping for coarse control
- Fast L-pulses for fine impulse
