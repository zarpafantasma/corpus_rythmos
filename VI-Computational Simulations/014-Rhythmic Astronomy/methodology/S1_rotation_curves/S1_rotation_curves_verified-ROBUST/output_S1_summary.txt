RTM MODEL
---------
v ∝ r^(1 - α/2)

Slope = 1 - α/2:
  α = 2.0 → slope = 0 (flat)
  α < 2.0 → slope > 0 (rising)
  α > 2.0 → slope < 0 (declining)

----------------
